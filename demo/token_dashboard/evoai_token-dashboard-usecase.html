// How It Works

From AI call to insight
in 4 steps

The pipeline is fully automated. Your team just opens the dashboard and reads the data.

01

AI workflow executes

An n8n workflow makes an AI API call — to any model, any provider. The call is tracked automatically as part of the workflow.

02

Log node captures metadata

A dedicated n8n log node captures model name, token counts (input + output), latency, cost, workflow name, and timestamp into a structured record.

03

Webhook delivers to dashboard

When "Show AI Activity Log" is clicked, a webhook fires to n8n — which fetches, aggregates, and returns the full analytics HTML payload in under 2 seconds.

04

Team reads the dashboard

Overview KPIs, token usage by model, cost charts, monthly summaries, and a full log table are rendered instantly — ready to share or screenshot.
